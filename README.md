# Dự Án Đầu Tiên Của Tôi 🚀

Chào mừng bạn đến với kho lưu trữ (repository) đầu tiên của mình trên GitHub!

## 📝 Giới thiệu
Đây là bước đi đầu tiên của mình trong thế giới lập trình và quản lý mã nguồn. Dự án này được tạo ra để mình làm quen với cách hoạt động của Git và GitHub.

## 🛠️ Công nghệ sử dụng
- **Git**: Quản lý phiên bản.
- **GitHub**: Lưu trữ mã nguồn.
- **Markdown**: Định dạng nội dung file này.

## 🎯 Mục tiêu
- [x] Tạo tài khoản GitHub.
- [x] Tạo kho lưu trữ đầu tiên.
- [ ] Tải file đầu tiên lên (Đang thực hiện).
- [ ] Học cách sử dụng Pull Request.
